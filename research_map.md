# Research Map (Detailed Strategic Structure)

## I. Core Research Track

### Humanoid Loco-Manipulation

Focus: Unified locomotion + manipulation in contact-rich environments.

Key Directions: - Whole-body policy learning - Force-adaptive control -
Contact modeling - Motion retargeting

------------------------------------------------------------------------

### Whole-Body Control

Focus: Coordinated multi-contact control for humanoids.

Key Directions: - Optimization-based WBC - Learning-based WBC -
Position-force unified frameworks - Stability-aware control

------------------------------------------------------------------------

### RL + MPC Hybrid Control

Focus: Combining learning flexibility with control stability.

Key Directions: - RL-augmented MPC - Data-driven MPC -
CLF/CBF-integrated learning - Stability guarantees

------------------------------------------------------------------------

### Sim2Real Transfer

Focus: Reliable deployment from simulation to hardware.

Key Directions: - Domain randomization - System identification - Physics
alignment - Robust policy transfer

------------------------------------------------------------------------
