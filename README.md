# Embodied Intelligence Research Roadmap

> Building a structured, long-term research map for Humanoid Robotics,\
> Loco-Manipulation, RL-MPC Hybrid Control, and Sim2Real Transfer.

------------------------------------------------------------------------

## 🎯 Research Vision

This repository is not a collection of starred projects.

It is a **strategic research map** focused on:

-   Humanoid Loco-Manipulation
-   Whole-Body Control
-   Reinforcement Learning + Model Predictive Control (RL-MPC)
-   Sim-to-Real Transfer
-   Scalable Robotic Intelligence

The goal is to build a coherent technical pathway toward advanced
humanoid intelligence systems.

------------------------------------------------------------------------

## 🧠 Core Research Track

### 1️⃣ Humanoid Loco-Manipulation

-   Unified locomotion + manipulation policies
-   Contact-rich interaction
-   Force-aware control

### 2️⃣ Whole-Body Control

-   Optimization-based WBC
-   Learning-based WBC
-   Position-force unified control

### 3️⃣ RL + MPC Hybrid

-   RL-augmented MPC
-   Data-driven MPC
-   Stability-aware learning control

### 4️⃣ Sim2Real Transfer

-   Domain randomization
-   Physics alignment
-   System identification

------------------------------------------------------------------------

## 🏗 Research Architecture

Core Intelligence ↓ Policy Learning (RL / Foundation Models) ↓ Hybrid
Control (RL + MPC) ↓ Whole-Body Coordination ↓ Sim2Real Transfer ↓
Hardware Deployment
